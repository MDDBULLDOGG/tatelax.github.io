Volygon

Volygon is my personal project I have been developing for several years. It is a multiplayer base building game built using Unity, Forge, MasterServerFramework, and Entitas ECS. It features realtime server authoritative base building with login and matchmaking.

Concept art was designed by myself and produced alongside a professional concept artist.

EmergeNYC

EmergeNYC is the most popular Emergency simulation game on Steam. It features real-time multiplayer using Unity and Photon up to 25 players.

I was a developer on this game responsible for feature implementation and design. One of our biggest challenge was managing CPU and memory resources due to the large scale of the level. We used Asset Bundles to take care of this problem.

PRO Engine

I was the Lead Developer for this project. The PRO Engine is a business process simulation tool. I designed and engineered this product to help 500 companies such as ING and AIG to help record and analyze the costs and resources required to complete a business process over a wide range of senarios and time periods.

Cave Crusade

Cave Crusade is one of my earliest projects. I designed and developed Cave Crusade as a 2D sidescrolling game for iOS and Android. This game was developed in Unity.

Miscellaneous

Sandboxr is a web based tool originally built in Unity. I provided consultation as well as the original proof-of-concept for this product. Sandboxr allows users to turn their in-game characters into real life figurines.
